# Antigravity IDE Cheatsheet

This document provides quick tips and commands for working with the Antigravity IDE and its AI features.

## AI Code Hints (Ghost Text)

When the editor suggests code as greyed-out "ghost text", you can manage it using these methods:

| Action | Interaction |
| :--- | :--- |
| **Dismiss Hint** | Press `Esc` to immediately clear the suggestion. |
| **Ignore & Skip** | Simply keep typing your own code; the hint will disappear or update. |
| **Undo Acceptance** | If you accidentally pressed `Tab`, use `Ctrl+Z` (Linux/Windows) or `Cmd+Z` (macOS). |
| **Settings** | Search for **"Completions"** or **"AI Hints"** in Settings to toggle or tune frequency. |
